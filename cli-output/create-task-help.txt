Usage: 0g-compute-cli create-task [options]

Create a fine-tuning task

Options:
  --key <key>                 Wallet private key, if not provided, ensure the
                              default key is set in the environment
  --provider <address>        Provider address for the task
  --model <name>              Pre-trained model name to use
  --data-size <size>          Token number of the dataset. Use calculate-token
                              command for the calculation
  --dataset <hash>            Hash of the dataset
  --config-path <path>        Fine-tuning configuration path
  --rpc <url>                 0G Chain RPC endpoint
  --ledger-ca <address>       Account (ledger) contract address
  --fine-tuning-ca <address>  Fine Tuning contract address
  --gas-price <price>         Gas price for transactions
  --max-gas-price <price>     Max gas price for transactions
  --step <step>               Step for gas price adjustment
  -h, --help                  display help for command
